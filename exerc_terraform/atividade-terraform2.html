☁️ Por que AWS CloudShell?
O Terraform é uma ferramenta de linha de comando. O CloudShell é um terminal Linux completo dentro do browser — sem instalar nada na sua máquina. Credenciais AWS já estão configuradas automaticamente. Arquivos persistem entre sessões (até 1GB). Tudo que você faz aqui é equivalente ao terminal local.
